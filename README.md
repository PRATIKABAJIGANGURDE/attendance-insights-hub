# Atharva Satellite Ground Station Attendance System

Welcome to the internal web portal for tracking and managing biometric attendance for the ASGS.

## Technology Stack

This project is built with:
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn-ui
- Appwrite Backend
- Inworld AI TTS (Premium Audio)

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Ensure your local `.env.local` accurately points to your internal Appwrite databases.
